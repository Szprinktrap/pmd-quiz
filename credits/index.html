This Website

This website was created by UsernameFodder. The markup, styling, and scripting were all done from scratch, because why not?

Questions

The original sources of the questions, answers, and nature descriptions used for this quiz are the following Pokémon Mystery Dungeon (PMD) games:

  • Pokémon Mystery Dungeon: Red/Blue Rescue Team
  • Pokémon Mystery Dungeon: Explorers of Time/Darkness
  • Pokémon Mystery Dungeon: Explorers of Sky

Response Point Values

The nature point values for responses were taken from the PMD Info Spreadsheet, created and verified by the researcher ShinxHijinx.

Modifications from the Source

All text was kept as close as possible to that of the original games, but minor modifications were made in a very limited number of cases. Response point values for certain questions were modified where appropriate. See the details page for more information.

Font

The lovely Mystery Dungeon font used for this site is a modified version of the Wonder Mail font, which was "painstakingly assembled by hand" by ShinxHijinx. I modified a few characters ('O', 'o', '8', '%', '♂', and '♀' to be specific) that weren't rendering properly in the original font.

Sprites

Sprites from the PMD: Explorers games were used. They were taken from the Spriter's Resource website. Direct links:

  • Pokémon Mystery Dungeon: Explorers of Time/Darkness
  • Pokémon Mystery Dungeon: Explorers of Sky
