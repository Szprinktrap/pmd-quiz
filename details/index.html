Quiz Details

Questions

  • Quiz questions are randomly selected from the pool of questions specified by the user during setup.
  • Questions that appear in more than one game are presented at most once.
  • Questions are presented in a random order, except for the "gender question," which always appears last.

Scoring

  • If you are unfamiliar with how the quizzes from the original games work, see this article.
  • In this quiz, response point values from Pokémon Mystery Dungeon: Explorers of Time/Darkness are halved from their original values for the purposes of scoring. This is to make their magnitudes more commensurate with those from the other games.
  • Unlike the original games, which use raw point totals for scoring, this quiz performs scoring using the lower bound of a 95% Wilson binomial proportion confidence interval, where proportions for each nature are calculated as the number of points scored divided by the hypothetical maximum number of points possible.
    • The scoring scheme balances rewarding a high proportion of points scored with the uncertainty of having a low sample size (i.e. a small total number of points possible).
    • The scoring scheme is the same one used by Reddit's "Best" comment sorting mode.
  • Scoring is performed multiple times; once over all quiz questions, and once for each individual game (using only points derived from questions from each particular game). The "all" score is used for deciding nature, while the "game" scores are used for determining nature descriptions (see next section).

Results and Starter Assignment

  1. The nature with the highest "all" score (see above section) is the nature presented to the user. Ties are broken randomly.
  2. After the nature has been decided, the "game" scores for the chosen nature are compared, and the user is assigned a game corresponding to the highest "game" score. Ties are broken randomly. The assigned game is used to determine the description of the nature and the actual assignment of starter Pokémon species.
  3. The starter Pokémon assignment also depends on the user's chosen gender. If "neither" was chosen, a gender is selected randomly for the purposes of assignment.
  4. The "other starters" result appears if different starters would have been assigned using different available games and/or genders.
    • Available genders are either male, female, or both in the case of a "neither" gender response.
    • Available games are based on the question sources chosen during setup.
  5. Natures in the detailed points breakdown are listed in descending order of "all" scores.

Modifications from the Source

All text was kept as close as possible to that of the original games, but minor modifications were made in a very limited number of cases.

  • A "neither" option was added to the final "gender question" of the quiz.
  • The Hasty nature's description from PMD: Rescue Team was changed to reference the "Next" button rather than the "A" button.
  • Certain questions from PMD: Explorers of Sky are almost identical to questions from PMD: Explorers of Time/Darkness, up to minor phrasing differences. This quiz keeps phrasing consistent between games. Which of the two variants were used in the quiz varies by question, depending on which I liked better.

Pokémon Super Mystery Dungeon

Pokémon Super Mystery Dungeon also has a personality quiz. Unfortunately, it has a different method of selecting the starter that is not directly tied to natures. Due to the incompatibility, questions from Pokémon Super Mystery Dungeon were not included in this quiz.
